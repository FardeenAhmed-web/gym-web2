# 💪Delhi-power-gym

A modern and fully responsive gym website built to showcase fitness services, workout plans, trainers, and memberships with a clean and attractive UI.

---

## 🚀 Features

✅ Fully Responsive Design  
✅ Modern User Interface  
✅ Smooth Scrolling Navigation  
✅ Membership Plans Section  
✅ Trainers Information Section  
✅ Contact Form  
✅ Mobile Friendly Layout  

---

## 🛠️ Technologies Used

- Tailwind css
- React js
- JavaScript
